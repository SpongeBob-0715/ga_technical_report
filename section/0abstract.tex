Long-horizon LLM agents are fundamentally bottlenecked by context: as interactions extend, useful information is overwhelmed by accumulated noise, undermining reasoning quality and operational efficiency. 
In this paper, we present GenericAgent (GA), a general-purpose agent system built around \textbf{context information density maximization}.
To operationalize this principle, GA integrates four core mechanisms: a minimal atomic tool set that enforces essential interactions, hierarchical on-demand memory that retrieves context by relevance, code-driven self-evolution that distills past trajectories into reusable capabilities, and structured browser extraction that compresses unstructured web content into task-relevant representations.
These mechanisms continuously suppress low-value context and preserve high-signal information during long-horizon execution. 
We conduct extensive experiments of GA across multiple evaluation dimensions, including task success rate, efficiency, tool usage, memory behavior, and web interaction, enabling a comprehensive comparison of long-horizon execution capability.
Results show that GA consistently improves the efficiency–performance trade-off, achieving higher task success at substantially lower token and interaction cost. This advantage persists across repeated runs, indicating stable, cost-efficient, and scalable agent behavior. 
We release GA as an open-source agent system at \href{https://github.com/lsdefine/GenericAgent/}{https://github.com/lsdefine/GenericAgent}.
